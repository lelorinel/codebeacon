# CodeBeacon

Use the `codebeacon` MCP tools for code navigation and context instead of manual file reads:

- `get_context` — start here for any exploration
- `find_definition` / `find_references` — jump to symbols
- `drill_package` — dig into a specific package
- `get_dependents` — find what uses a symbol

Prefer these over `Read`/`grep` for navigating this codebase.
